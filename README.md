# SieveOfEratosthenes-C-Threads
Sieve of Eratosthenes algotithm with usage of threads in C, for my Uni project.

Compilation:
```
gcc -Wall -pthread -lm main.c
```


Execution:
```
./a.out [upper limit of prime numbers] [number of threads] [output file]
```
